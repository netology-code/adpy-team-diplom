# Как настроить группу для бота и получить токен

## Для создания токена для бота нужно
1. Создать группу в VK
2. Зайти в Управление -> Дополнительно -> Работа с API. Создать ключ.
3. Нажать "Показать". Скопировать ключ в Вашу программу. Это будет вашим токен для запросов от группы.
<img width="961" height="468" alt="image" src="https://github.com/user-attachments/assets/44866d4b-b886-45bf-b8e7-28497e38c7ee" />

4. Включить возможность писать сообщения в группу. Управление -> Сообщения -> Настройки для бота -> Сообщения сообщества: включить.
5. Настройки бота. Возможности бота: Включены
